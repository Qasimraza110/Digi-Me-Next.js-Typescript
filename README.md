# 🚀 Digi-Me | QR-Based Digital Profile Platform (MVP)

**Digi-Me** is a digital identity and networking platform that lets users create, share, and connect through dynamic QR codes.  
This MVP version focuses on providing seamless profile creation, secure authentication, and QR-based sharing — making networking smarter and paperless.

---

## 🌟 Key Features
- **User Authentication:** Secure email/password signup, Google login, and password reset functionality.  
- **Profile Management:** Users can create and update their digital profiles with:
  - Social links (Facebook, Instagram, LinkedIn, WhatsApp)
  - Business/Website URLs
  - Contact details (Phone, Email)
  - Bio and profile picture upload  
- **Dynamic QR Code Generation:** Every profile is linked to a unique, shareable QR code that opens a public profile page.  
- **Public Profile View:** Anyone scanning the QR can access a clean, public page showing all user details and links.  
- **Save Profile Feature:** Logged-in users can save scanned profiles to their dashboard for quick access and networking.  
- **Free & Unlimited:** QR codes are generated dynamically without any limits or payment requirements.

---

## 🧩 Tech Stack
**Frontend:** TypeScript
**Backend:** Node.js, Express.js
**Database:** MongoDB   
**Authentication:**  JWT  
**QR Code Generation:** QRCode.js
